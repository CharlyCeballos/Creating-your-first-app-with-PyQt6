## Course division

The course is divided by modules (letters) and sections (numbers)


## Virtual Environment

If you don't wanna install globally PyQt6 you can create an Virtual Environment

```bash
  python -m venv .venv
  .venv\Scripts\activate
  pip install -r requirements.txt
```
